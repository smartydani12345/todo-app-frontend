# Frontend Engineer Skill

## Purpose
Provides Next.js UI development and animations capabilities for the Todo Evolution Hackathon application.

## Functions
- Create Next.js 16+ components
- Implement Tailwind CSS styling
- Apply Framer Motion animations
- Build responsive UI elements
- Develop accessibility features
- Implement dark/light mode
- Add loading states and error toasts
- Create voice command integration using Web Speech API
- Implement i18n support for EN/UR language toggle

## Usage
```
/sp.frontend-engineer [ui-feature-request]
```

## Constraints
- Follow mobile-first responsive design
- Maintain performance standards
- Ensure accessibility compliance
- Use consistent styling patterns
- Implement comma-separated tag input with validation and auto-suggestions
- Use specific priority badge colors: high (bg-red-100 text-red-800), medium (bg-yellow-100 text-yellow-800), low (bg-green-100 text-green-800)
- Implement 300ms debounce for search functionality
- Default to light mode but detect system preference with option to sync with system settings
