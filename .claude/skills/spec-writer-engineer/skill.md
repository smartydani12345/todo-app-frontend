# Spec Writer Engineer Skill

## Purpose
Provides specification, planning, and task definition capabilities for the Todo Evolution Hackathon application.

## Functions
- Create project specifications
- Develop implementation plans
- Define tasks and requirements
- Clarify ambiguous requirements
- Create architecture decision records
- Maintain documentation
- Organize project artifacts
- Ensure specification/plan/tasks lineage

## Usage
```
/sp.spec-writer-engineer [spec-planning-request]
```

## Constraints
- Follow Spec-Driven Development
- Create clear, testable specs
- Maintain consistent documentation
- Follow template structures
- Enable proper implementation handoff
- Ensure constitution compliance throughout the project lifecycle
