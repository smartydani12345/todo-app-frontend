# Database Engineer Skill

## Purpose
Provides SQLModel/Neon database models and queries capabilities for the Todo Evolution Hackathon application.

## Functions
- Create SQLModel models
- Implement database queries
- Manage Neon PostgreSQL connections
- Design database schemas
- Handle migrations
- Implement async queries
- Filter user-specific data
- Create performance indexes

## Usage
```
/sp.database-engineer [database-feature-request]
```

## Constraints
- Use SQLModel ORM
- Connect to specified Neon DB
- Filter by authenticated user_id
- Implement async queries
- Maintain data integrity
- Create proper indexes for common filtering/sorting operations
- Use async SQLAlchemy operations with proper connection pooling
