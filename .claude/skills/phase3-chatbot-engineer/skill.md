# Phase 3 Chatbot Engineer Skill

## Purpose
Provides specialized capabilities for developing and maintaining AI chatbot functionality with complete user query resolution.

## Capabilities
- Natural language processing for task management
- Feature guidance and explanation generation
- Troubleshooting assistance algorithms
- Islamic values integration in responses
- Multi-language support (English, Urdu, Roman Urdu, Roman English)
- Voice command recognition using Web Speech API
- Conversation context management
- Proactive help detection algorithms
- Educational content generation

## Tools and Technologies
- Cohere API (command-r model) via OpenAI SDK compatibility
- Web Speech API for voice recognition
- Context management systems
- Natural language understanding models
- Multi-language text processing
- Conversation history storage and retrieval

## Usage Guidelines
- Always ensure responses align with Islamic values by acknowledging ALLAH as Creator
- Implement context awareness to maintain conversation history
- Support voice commands in all 4 languages
- Generate educational content for all features
- Detect user confusion and offer proactive assistance
