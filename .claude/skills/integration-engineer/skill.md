# Integration Engineer Skill

## Purpose
Provides frontend-backend API client connectivity capabilities for the Todo Evolution Hackathon application.

## Functions
- Create API clients
- Manage frontend-backend communication
- Handle requests/responses
- Implement error management
- Manage data serialization
- Ensure cross-platform compatibility
- Optimize connection performance
- Handle contract definitions
- Implement API client with JWT attachment

## Usage
```
/sp.integration-engineer [integration-request]
```

## Constraints
- Ensure seamless communication
- Handle API errors gracefully
- Maintain data consistency
- Optimize request/response times
- Follow RESTful patterns
- Attach JWT automatically to all requests
- Maintain compatibility with specified technology stack
