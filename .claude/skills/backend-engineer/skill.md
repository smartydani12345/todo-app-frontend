# Backend Engineer Skill

## Purpose
Provides FastAPI routes and business logic capabilities for the Todo Evolution Hackathon application.

## Functions
- Create FastAPI routes
- Implement business logic
- Design RESTful APIs
- Handle async operations
- Implement Pydantic validation
- Manage error handling
- Generate API documentation
- Create reusable service layer for Phase 3 compatibility

## Usage
```
/sp.backend-engineer [backend-feature-request]
```

## Constraints
- Follow RESTful API design principles
- Implement async routes
- Use Pydantic for validation
- Handle errors gracefully
- Implement proper validation rules for title length, tag count, and enums
- Return 422 on invalid input
- Create reusable service layer for Phase 3 chatbot compatibility
